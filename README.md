My personal zsh setup.
Made for macOS. Will need some tweaking for Linux.

Installs Oh My ZSH, PL10k as well as other useful CLI things like fonts and brew packages.
Oh and some plugins.

1.
```
git clone https://github.com/iamlite/myzsh
```
2.
```
cd myzsh
```
3.
```
./setup.zsh
```
